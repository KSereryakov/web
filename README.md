# KSerebryakov.github.io
# 🔍 Диагностика сети и блокировок

Инструмент для проверки доступности SNI доменов и анализа блокировок мобильного интернета в России.

## 🚀 Использование

Откройте страницу в браузере на мобильном интернете:
**[https://ваш-username.github.io/network-diagnostics/](https://ваш-username.github.io/network-diagnostics/)**

## 📋 Что проверяется

### 1. Информация о клиенте
- Публичный IP адрес
- Тип сети (4G/5G)
- Скорость соединения и задержка (RTT)
- Оператор связи

### 2. IP сервера в whitelist
- Проверка попадания IP VPN сервера в whitelist подсети операторов
- Если IP в whitelist - SNI spoofing не нужен

### 3. Диагностика блокировок оператора
- HTTP/2 поддержка (обязательна для Reality протокола)
- WebSocket доступность
- DNS резолвинг через Cloudflare
- UDP/STUN доступность

### 4. Проверка SNI доменов
- Проверка ~890 доменов из [russia-mobile-internet-whitelist](https://github.com/hxehex/russia-mobile-internet-whitelist)
- Параллельная проверка батчами по 30 доменов
- Живое отображение текущего проверяемого домена
- Экспорт результатов в JSON

## 📊 Результаты

После завершения диагностики вы получите:
- Список доступных SNI доменов для Reality протокола
- Процент доступности доменов
- Информацию о блокировках оператора
- Рекомендации по настройке VPN

## 💾 Экспорт

Нажмите кнопку "Экспортировать результаты" для сохранения полного отчета в JSON формате.

## 🛠 Технологии

- Vanilla JavaScript (без зависимостей)
- GitHub whitelist API
- WebRTC для определения локального IP
- Network Information API для характеристик сети

## 📝 Примечания

- Проверка должна выполняться на **мобильном интернете** (не WiFi)
- Результаты зависят от вашего оператора и региона
- Проверка занимает 2-3 минуты
